# validation
1. **Защитить программу от некорректных данных**, которые могут вызвать ошибки или неправильную работу.
2. **try...except** позволяет перехватить ошибку и обработать её, не завершая программу аварийно.
3. **else** выполняется если в try не было ошибок. finally выполняется всегда — и при ошибке и без неё.
4. **ValueError** — возникает когда строка не может быть преобразована в число.
5. **Используется while True** — бесконечный цикл. Оператор break прерывает его когда ввод корректен.
6. **Регулярные выражения** позволяют проверить что строка соответствует нужному формату — например ИНН из 12 цифр или корректный email.
